ОписаниеБазы

Класс описания базы данных

Свойства:

Идентификатор

Идентификатор базы как он задан в конфигурационном файле.

СтрокаПодключения

Строка подключения в виде File='D:\base' или Srvr='server';Ref='base'. Соответствует ключу conn.

Пользователь

Имя пользователя (не указывается, если в базе нет пользователей). Соответствует ключу user, необязательный.

Пароль

Пароль пользователя (не указывается, если не указан пользователь). Соответствует ключу pwd, необязательный.

ВерсияПлатформы

Версия платформы в виде 8.3, 8.3.10, 8.3.10.2667. По умолчанию - 8.3. Соответствует ключу version, необязательный.

КлючРазрешения

Ключ разрешения (параметр /uc), который будет использоваться при блокировке сеансов. Соответствует ключу key, необязательный.

ИмяКоннектора

Имя COM-коннектора, если он зарегистрирован под другим именем. Может использоваться в случае, когда необходимо иметь несколько зарегистрированных коннекторов разных версий. По умолчанию - V83.COMConnector (или V82.COMConnector, в зависимости от платформы). Соответствует ключу comconnector, необязательный.

ИмяСервераSQL

Указывается в случае, когда имя сервера СУБД отличается от имени сервера 1с. По умолчанию определяется из строки соединения. Соответствует ключу sql-server, необязательный.

ИмяБазыSQL

Указывается в случае, когда имя базы на сервере СУБД отличается от имени базы на сервере 1с. По умолчанию определяется из строки соединения. Соответствует ключу sql-base, необязательный.

ПользовательХранилища

Пользователь хранилища (не указывается, если база не подключена к хранилищу). Соответствует ключу repo-user, необязательный.

ПарольХранилища

Пароль хранилища (не указывается, если база не подключена к хранилищу). Соответствует ключу repo-pwd, необязательный.

ПриСозданииОбъекта

Конструктор.

Параметры:

  • ИД - Строка - Идентификатор базы как он задан в конфигурационном файле.
  • Значения - Соответствие - Значения, считанные из конфигурационного файла.

ПолучитьКонфигуратор

Возвращает объект управления конфигурацтором.

Возвращаемое значение:

  • УправлениеКонфигуратором

ПолучитьCOMСоединение

Формирует строку подключения для COM-объекта.

Возвращаемое значение:

  • Строка

ПолучитьОбъектОбработкиДействий

Возвращает объект внешней обработки (см. src/templates/ДействияСБазойШаблон.epf).

Параметры:

  • Соединение - COMОбъект - COM-соединение (COMConnector.Connect).
  • ПутьКОбработке - Строка - Путь к внешней обработке.

Возвращаемое значение:

  • COMОбъект

ПолучитьОбъектОбработкиБлокировки

Возвращает объект внешней обработки (см. src/БлокировкаСеансов.epf).

Параметры:

  • Соединение - COMОбъект - COM-соединение (COMConnector.Connect).

Возвращаемое значение:

  • COMОбъект

ПолучитьВерсиюКонфигурации

Версию конфигурации из метаданных базы. Используется COM-соединение.

Возвращаемое значение:

  • Строка

ЭтоФайловаяБаза

Определяет по строке подключения, является ли база файловой.

Возвращаемое значение:

  • Булево

ПолучитьПутьКБазе

Определяет каталог базы по строке подключения.

Возвращаемое значение:

  • Строка, Неопределено - Неопределено - если база серверная.

ПолучитьИмяСервера

Определяет имя сервера по строке подключения.

Возвращаемое значение:

  • Строка, Неопределено - Неопределено - если база файловая.

ПолучитьИмяБазы

Определяет имя базы на сервере по строке подключения.

Возвращаемое значение:

  • Строка, Неопределено - Неопределено - если база файловая.
