ОбновитьКонфигурацию

Обновляет конфигурацию из файла поставки или каталога дистрибутива.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
  • ИмяМассивФайлов - Строка, Массив - Указывается либо полное имя файла поставки, либо каталог, в котором содержится файл поставки (1Cv8.cf, 1Cv8.cfu). В случае указания массива будет произведено последовательное обновление из каждого файла или каталога в массиве.
  • Отказ - Булево - Признак отказа от продолжения действий.

ЗапуститьВРежимеПредприятия

Запускает базу в режиме предприятия. Возможен запуск с произвольным параметром. Возможен запуск с открытием внешней обработки.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
  • ПараметрЗапуска - Строка - Произвольный параметр запуска (/C) (необязательный).
  • ЗапуститьОбработку - Строка - Путь к внешней обработке (необязательный).
  • Отказ - Булево - Признак отказа от продолжения действий.

ВыполнитьОбработкуПодготовки

Запускает внешнюю обработку, содержащую произвольные действия с базой (см. src/templates/ДействияСБазойШаблон.epf). Используется COM-соединение.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
  • ПутьКОбработке - Строка - Путь к внешней обработке.
  • ИмяСценария - Строка - Имя сценария, заданного в обработке.
  • Отказ - Булево - Признак отказа от продолжения действий.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
