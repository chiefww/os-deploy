Класс описания учетной записи почты

Свойства:

Идентификатор

Идентификатор учетной записи почты как он задан в конфигурационном файле.

АдресSMTP

Адрес smtp-сервера. Соответствует ключу smtp.

ПортSMTP

Порт smtp-сервера. Соответствует ключу port.

ПользовательSMTP

Пользователь smtp-сервера. Соответствует ключу user.

ПарольSMTP

Пароль пользователя smtp-сервера. Соответствует ключу pwd.

ИспользоватьSSL

Признак испоьзования защищенного соединения. По умолчанию Истина. Соответствует ключу ssl, необязательный.
