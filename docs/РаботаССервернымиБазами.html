РаботаССервернымиБазами

Модуль для работы с серверными базами и кластерами серверов

Методы:

  • ОтключитьСеансы
  • УстановитьБлокировкуРегламентныхЗаданий

ОтключитьСеансы

Отключает сеансы и соединения базы данных.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
  • ИмяАдминистратора - Строка - Имя администратора кластера (необязательный).
  • Пароль - Строка - Пароль администратора кластера (необязательный).

УстановитьБлокировкуРегламентныхЗаданий

Устанавливает признак блокировки регламентных заданий в базе данных.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
  • Блокировка - Булево - Значение признака блокировки.
  • ИмяАдминистратора - Строка - Имя администратора кластера (необязательный).
  • Пароль - Строка - Пароль администратора кластера (необязательный).
