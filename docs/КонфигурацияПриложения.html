ПриСозданииОбъекта

Конструктор.

Параметры:

  • пПутьКФайлу - Строка - Путь к файлу конфигурации.

ПолучитьБазу

Получает описание базы данных из конфигурационного файла.

Параметры:

  • Имя - Строка - Имя базы данных как оно задано в конфигурационном файле.

Возвращаемое значение:

  • ОписаниеБазы, Неопределено - Неопределено, если база с таким ключом не найдена в файле.

ПолучитьПараметр

Получает параметр из конфигурационного файла.

Параметры:

  • Имя - Строка - Имя параметра как оно задано в конфигурационном файле.

Возвращаемое значение:

  • Произвольный - Неопределено, если параметр с таким ключом не найден в файле.

ПолучитьАдресПочты

Получает описание учетной записи электронной почты из конфигурационного файла.

Параметры:

  • Имя - Строка - Имя параметра как оно задано в конфигурационном файле.

Возвращаемое значение:

  • ОписаниеУчетнойЗаписиПочты, Неопределено - Неопределено, если учетная запись с таким ключом. не найдена в файле

ПолучитьFTP

Получает описание учетной записи ftp-сервера из конфигурационного файла.

Параметры:

  • Имя - Строка - Имя параметра как оно задано в конфигурационном файле.

Возвращаемое значение:

  • ОписаниеFTPСоединения, Неопределено - Неопределено, если подключение с таким ключом не найдено в файле.
