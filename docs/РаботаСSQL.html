РаботаСSQL

Модуль для работы с базами средствами MSSQL

Методы:

  • СоздатьБэкап
  • ЗагрузитьБэкап
  • ПолучитьОбъектУправленияSQL

СоздатьБэкап

Создает бэкап базы.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
  • ИмяПользователя - Строка - Имя учетной записи MSSQL.
  • Пароль - Строка - Пароль учетной записи MSSQL.
  • ИмяФайла - Строка - Путь к файлу бэкапа.

ЗагрузитьБэкап

Загружает бэкап базы.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
  • ОписаниеБазыИсточника - ОписаниеБазы - В случае, когда бэкап загружается в другую базу, нужно указать базу-источник (необязательный).
  • ИмяПользователя - Строка - Имя учетной записи MSSQL.
  • Пароль - Строка - Пароль учетной записи MSSQL.
  • ИмяФайла - Строка - Путь к файлу бэкапа.

ПолучитьОбъектУправленияSQL

Получает объект для работы с MSSQL.

Параметры:

  • ОписаниеБазы - ОписаниеБазы - Объект описания базы.
  • ИмяПользователя - Строка - Имя учетной записи MSSQL.
  • Пароль - Строка - Пароль учетной записи MSSQL.

Возвращаемое значение:

  • УправлениеMSSQL
