ОписаниеFTPСоединения

Класс описания FTP соединения

Свойства:

Идентификатор

Идентификатор учетной записи ftp как он задан в конфигурационном файле.

АдресСервера

Адрес ftp-сервера. Соответствует ключу ftp.

Порт

Порт ftp-сервера. Соответствует ключу port.

ИмяПользователя

Пользователь ftp-сервера. Соответствует ключу user.

Пароль

Пароль пользователя ftp-сервера. Соответствует ключу pwd.

Методы:

  • ПриСозданииОбъекта
  • ПолучитьСоединение

ПриСозданииОбъекта

Конструктор.

Параметры:

  • ИД - Строка - Идентификатор учетной записи ftp как он задан в конфигурационном файле.
  • Значения - Соответствие - Значения, считанные из конфигурационного файла.

ПолучитьСоединение

Возвращает объект FTP-Соединения.

Возвращаемое значение:

  • FTPСоединение
