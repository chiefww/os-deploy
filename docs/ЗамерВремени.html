ЗамерВремени

Класс замера времени выполнения действий

Методы:

НачатьЗамер

Запускает замер времени, устанавливает дату начала отсчета.

Параметры:

  • Дата - Дата - Дата и время начала отсчета, если не указано, используется текущая дата (необязательный).

ЗакончитьЗамер

Останавливает замер времени, устанавливает дату окончания отсчета.

Параметры:

  • Дата - Дата - Дата и время окончания отсчета, если не указано, используется текущая дата (необязательный).

ПолучитьРезультат

Возвращает разницу в секундах между окончанием и началом отсчета.

Возвращаемое значение:

  • Число

ПолучитьРезультатСтрокой

Возвращает разницу окончанием отсчета и началом отсчета в виде форматированной строки.

Возвращаемое значение:

  • Строка

ФорматироватьРезультат

Форматирует количество секунд в читаемый вид.

Параметры:

  • Результат - Чтсло

Возвращаемое значение:

  • Строка
